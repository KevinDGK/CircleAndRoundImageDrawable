# CircleAndRoundImageDrawable
利用Drawable实现圆形图片和圆角矩形图片，即可以作为ImageView的背景，也可以作为其他View的背景，使用方式十分灵活！
#代码
内部实现方式是是使用BitmapShader+重写Drawable
